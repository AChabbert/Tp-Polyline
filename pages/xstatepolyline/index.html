Exercice "Polyline"

Mode d'emploi

Tracer une "polyline" (suite de points interconnectés)

  • MOUSECLICK ajoute un point (jusqu'au maximum MAX fixé)
  • MOUSEMOVE déplace le point provisoire
  • Backspace efface le dernier point ajouté (sauf le premier)
  • Enter enregistre la polyline (2 ≤ nombre de points ≤ MAX)
  • Ecape abandonne le tracé de la polyline
Ce code utilise la bibliothèque Konva pour dessiner dans un canvas javascript. Il démontre l'utilisation de la bibliothèque xState pour implémenter un statechart.
